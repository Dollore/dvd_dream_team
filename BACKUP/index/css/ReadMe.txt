CSS jest poukładany według reguł SMACSS

https://css-tricks.com/methods-organize-css/#article-header-id-1

base - podstawa, uniwersalne zasady typu: a:link itp.
layout - zasady ułożenia strony (zasady grid oraz podstawowe rzeczy jak background-color, color)
module - cała reszta, zasady do poszczególnych modułów(eg. navbar, header, main itp.)

child_ - dziecko głównego kontenera(container)
content_ - wszytko co moduł zmienia (moduł)

Staram się zachować porządek, w razie niejednolitości proszę powiedzieć.
